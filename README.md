This project is a console-based Food Ordering System developed in Java. The application provides a menu where users can select different food items such as Pizza, Burger, French Fries, and Sandwich. After selecting the item, users can choose from multiple size options (Small, Medium, Large), each with a different price. The user then enters the desired quantity, and the program calculates the total price based on the selected item, size, and quantity.

The program also accepts flexible user input for selecting sizes (such as number, name, or price). After processing the order, it displays the order details and the total bill to the user.

Features -
1) Menu-based food selection
2) Multiple size options with different prices
3) Quantity selection
4) Automatic total bill calculation
5) User-friendly console interaction
